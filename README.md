# The CypherHub Blog

The CypherHub Blog is a modern, responsive blogging platform built with Next.js. It features a clean, dark-themed UI, an admin interface for managing content, and robust features for both readers and authors.

## üöÄ Tech Stack

- **Framework:** [Next.js 15+](https://nextjs.org/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Components:** [Radix UI](https://www.radix-ui.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/), [React Icons](https://react-icons.github.io/react-icons/)
- **Forms:** [React Hook Form](https://react-hook-form.com/) + [Zod](https://zod.dev/)
- **Notifications:** [Sonner](https://sonner.emilkowal.ski/), [React Hot Toast](https://react-hot-toast.com/)

## ‚ú® Features

- **Modern UI/UX:** Sleek dark-themed design with smooth animations.
- **Notes & Articles:** Two distinct content types for short updates and long-form articles.
- **Rich Text Support:** Syntax highlighting for code blocks.
- **Responsive Design:** Optimized for all device sizes.
- **Admin Interface:** (Implied) Tools for creating and managing posts.

